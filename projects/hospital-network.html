Hospital Network Infrastructure

Cisco Packet Tracer Project

Overview

This project demonstrates the design and implementation of a hospital network infrastructure using Cisco Packet Tracer. The network is designed to support communication between multiple departments while maintaining security, scalability, and centralized network management.

Objectives

  • Design a scalable hospital network.
  • Separate departments using VLAN.
  • Improve network security.
  • Provide centralized network services.
  • Ensure communication between all departments.

Features

Network Features

  • ✔ VLAN Segmentation
  • ✔ Inter VLAN Routing
  • ✔ DHCP Server
  • ✔ DNS Server
  • ✔ Layer 3 Switching
  • ✔ Static Routing
  • ✔ Centralized Management
